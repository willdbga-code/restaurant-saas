# Project Architecture: Restaurant SaaS
Updated: 2026-04-18 22:22:56

## Overview
A multi-tenant SaaS for restaurants, built with Next.js (App Router), Firebase (Firestore, Auth, Functions), and Tailwind CSS.

## Key Features
- **POS (PDV)**: Table management and order entry.
- **KDS (Kitchen Display System)**: Dedicated views for Kitchen and Bar.
- **Digital Menu**: QR-code based menu for customers.
- **Admin Dashboard**: Revenue reports, product management, and staff invitations.

## Technical Stack
- **Frontend**: Next.js 16+, TypeScript, Tailwind CSS.
- **Backend**: Firebase Cloud Functions.
- **Database**: Firestore.
- **Memory**: MemPalace (Local Vector Database).
